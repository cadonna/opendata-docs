---
title: Ingest
description: Highly-available ingestion on object storage
---

Ingest is a simple data integration system that leverages object storage to reliably ingest data into a
database without a stateful broker.
Instead data producers write directly to object storage and databases read the data directly from object storage.
A single binary manifest in object storage coordinates the reads and writes using queue semantics.
Because the data and the manifest live in object storage, Ingest is stateless and horizontally scalable.
Object storage takes care of durability and availability.
